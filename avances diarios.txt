✅ Resumen del Trabajo Realizado
1️⃣ Configuración del Proyecto
✔️ Creamos un proyecto en Vite con React.
✔️ Instalamos dependencias y configuramos el entorno de desarrollo.

2️⃣ Implementación del Navbar
✔️ Agregamos un Navbar con enlaces de navegación: Inicio, Clientes y Configuración.
✔️ Creamos el archivo Navbar.jsx en src/components/.
✔️ Definimos los estilos del Navbar en SASS, guardándolos en src/styles/Navbar.scss.
✔️ Solucionamos un error de importación en los estilos cambiando la ruta de import.

3️⃣ Configuración de SASS y Estilos Globales
✔️ Instalamos y configuramos SASS (npm install sass).
✔️ Creamos un archivo global main.scss para manejar colores y estilos globales.
✔️ Definimos una nueva paleta de colores más moderna.
✔️ Aplicamos los colores al Navbar y la estructura del sitio.
✔️ Reemplazamos @import por @use en los archivos SASS para evitar advertencias.

4️⃣ Creación de las Rutas en React Router
✔️ Configuramos React Router en App.jsx.
✔️ Agregamos las rutas para Home, Clientes y Configuración.
✔️ Creamos la vista Configuracion.jsx para que la ruta /configuracion funcione.
✔️ Solucionamos un error de "No routes matched location '/configuracion'".
